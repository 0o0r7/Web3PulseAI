# Submission Checklist

This checklist will help you ensure that you have all the required artifacts for the Contra-Hackathon submission.

-   [ ] **Prompt:** A plain English description that clearly defines the app idea and desired features.
-   [ ] **App Build:** Generated codebase and deployments (web/mobile apps, backend systems) based on the prompt.
-   [ ] **Supporting Data:**
    -   [ ] Screenshots of the application.
    -   [ ] A demo video of the application.
    -   [ ] Documentation detailing:
        -   [ ] App usage, flows, and features designed by the AI.
        -   [ ] Customizations, integration details, and business logic.
-   [ ] **Submission Artifacts:**
    -   [ ] A link to the live application/demo.
    -   [ ] A downloadable package of the application.
    -   [ ] Instructions for the judges to test and verify the application.
